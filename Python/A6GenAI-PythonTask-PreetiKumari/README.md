## Tasks Completed

### Task 1: Safe Division Utility

Created a program that:

* Takes numerator and denominator as input
* Handles invalid inputs
* Handles division by zero
* Uses `try`, `except`, `else`, and `finally`

Exceptions used:

* `ValueError`
* `ZeroDivisionError`

---

### Task 2: Bill Calculator with Error Handling

Created a bill calculator that:

* Iterates through a list of prices
* Skips invalid values
* Rejects negative prices
* Calculates the final bill

Concepts used:

* `TypeError`
* `ValueError`
* `raise`

---

### Task 3: Custom Exception - Age Validator

Created an age validation program that:

* Accepts user age
* Allows ages between 1 and 120
* Raises an exception for invalid ages

Concepts used:

* `raise`
* `ValueError`

---

### Task 4: File Reader with Exception Handling

Created a program that:

* Reads the first 3 lines of a file
* Handles missing files
* Handles permission issues

Exceptions used:

* `FileNotFoundError`
* `PermissionError`

---

### Task 5: Mini Program - Safe Shopping Cart

Created a shopping cart program that:

* Accepts multiple prices
* Rejects invalid values
* Rejects negative prices
* Calculates total items and total bill

Concepts used:

* `while` loop
* `try`
* `except`
* `raise`
